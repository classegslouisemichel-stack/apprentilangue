📖 Guide utilisateur

Ce guide est destiné aux enseignantes qui utilisent Apprentilangue. Pas besoin de connaissances informatiques particulières !
🔍
Conseil avant de commencer — ajustez le zoom de votre navigateur

Pour que les diapositives occupent bien tout votre écran, adaptez le zoom à votre affichage :

  • Firefox / Chrome : maintenez Ctrl et tournez la roulette de la souris
  • Tous navigateurs : Ctrl + + pour agrandir, Ctrl + − pour réduire, Ctrl + 0 pour revenir à la normale
  • Mac : remplacez Ctrl par Cmd

Le bon zoom est celui où les diapositives remplissent bien l'écran sans barre de défilement.

Qu'est-ce qu'Apprentilangue ?

Un outil pour créer des diaporamas interactifs pour travailler le vocabulaire avec vos élèves. Pour chaque mot, l'application affiche la définition, l'image du mot, les sons lettre par lettre, le genre et la nature. Tout se passe dans votre navigateur — rien à installer.

1. Lancer un diaporama

  1. Repérez la série sur l'écran d'accueil
  2. Cliquez sur ▶ Jouer
  3. Choisissez l'affichage des lettres : Scriptes (habituel), Capitales, ou Cursive
  4. Cliquez sur ▶ Lancer

2. Naviguer dans le diaporama

ActionComment faire
AvancerClic sur la diapositive — ou touche → ou Espace
ReculerTouche ←
QuitterTouche Échap

Diapositive 1 — chaque clic révèle : définition → image → image du son → lettre. Le dernier clic passe à la diapositive 2.

Diapositive 2 — genre (MASCULIN / FÉMININ), nature (NOM / VERBE…), écriture cursive et scripte.

Sommaire — cliquez directement sur un mot pour y accéder.

3. Créer une nouvelle série

  1. Cliquez sur + Nouvelle série
  2. Remplissez le nom, la couleur, la police et la méthode (EO ou BM)
  3. Cliquez sur Créer

4. Ajouter des mots à la main

  1. Dans l'éditeur de série, cliquez sur + Ajouter un mot
  2. Remplissez : mot (en majuscules), définition, genre, nature, image
  3. Cliquez sur Enregistrer
💡 Les sons sont générés automatiquement. Vous pouvez les corriger manuellement si besoin.

5. Importer depuis Excel

La méthode la plus rapide pour de nombreux mots. Votre fichier doit avoir ces colonnes :

Col.ContenuExemple
ANuméro1
BMot (majuscules)AMOUREUX
CDéfinitionC'est quelqu'un qui aime…
DGenremasc / fem / none
ENaturenom / verbe / adjectif / autre
FImage (optionnel)laisser vide
GSegmentation (optionnel)a-mou-reux
HLiaisons (optionnel)ou;eu
ICouleurs (optionnel)ou=rouge

Pour importer : ⬆️ Import → 📥 Importer depuis Excel → choisir le fichier → remplir le nom de série et le numéro de dossier → Importer.

6. Corriger les sons d'un mot

  1. Dans l'éditeur, cliquez sur le mot
  2. Section Sons : modifiez son, image, liaison de chaque groupe
  3. Utilisez ⊕ pour fusionner deux groupes, ⊗ pour séparer
  4. Cliquez sur Enregistrer

7. Partager avec vos collègues

  1. ⬇️ Export → 📤 Générer series_partagees.json
  2. Lancez MAJ_VERSION.bat (Windows) ou MAJ_VERSION.sh (Mac)
  3. Déposez les deux fichiers sur GitHub

Vos collègues voient les nouvelles séries automatiquement, sans Ctrl+F5.

Pour supprimer une série partagée : bouton 🗑 sur sa carte → déposer le fichier téléchargé sur GitHub.

8. Questions fréquentes

❓ Les nouveautés n'apparaissent pas
Attendez 5 minutes (mise à jour automatique) ou faites Ctrl + F5.
❓ Mes séries ont disparu
Elles sont dans votre navigateur. Exportez-les régulièrement (bouton ⬇️ sur chaque carte) pour les sauvegarder.
❓ Les images ne s'affichent pas
Vérifiez que l'image est sur GitHub dans le bon dossier et que le chemin correspond exactement (majuscules, accents…).
❓ Un mot s'affiche en CAPITALES au lieu de scriptes
Au lancement, choisissez Scriptes — les lettres accentuées seront converties automatiquement.
❓ Quelque chose ne fonctionne pas
Essayez F5 puis Ctrl + F5. Si le problème persiste, notez le nom du mot et de la série et contactez votre coordinateur.

9. Utiliser le quiz

Le quiz affiche les images des mots en très grand format (2 par page) pour une révision rapide en classe.

  1. Sur la carte d'une série, cliquez sur 🎯 Quiz
  2. Choisissez jusqu'à quel numéro de mot inclure (ex : 1 à 12 si les élèves ont appris 12 mots)
  3. Cliquez sur ▶ Lancer ce quiz

Dans le quiz : chaque image est masquée, cliquez dessus pour la révéler (mot affiché en dessous). Utilisez → ou le bouton Suivant pour passer à la paire suivante.

💡 Méga-quiz : cochez plusieurs séries dans la fenêtre de configuration pour mélanger des mots de différentes séries. Idéal pour une révision globale !
